什么是独立博客？

独立博客就是区别于平台博客（如简书、知乎、豆瓣）的一种方案，特点即：

域名独立（地址是什么）
空间独立（部署到哪里）
内容独立（自己即审核）
通俗来说就是 住自己的房子 与 寄人篱下 的区别。所以，我们为什么需要一个独立博客呢？

希望数据属于自己，随时可以更换技术方案，而非交给某个平台，与某个平台共存亡。
希望自己设计，或不满足于平台支持的格式，希望有更多的内容表达方式。
希望有自己的短域名，任何人输入域名就进入自己的空间。

插件

npm install hexo-generator-searchdb --save   #用于搜索
npm install hexo-generator-json-content@2.2.0 --save  #添加本地搜索
npm install hexo-generator-feed --save       #用于生成 RSS 订阅
npm uninstall hexo-generator-index --save     
npm install hexo-generator-index-pin-top --save  #用于文章置顶

常用命令：

npm install hexo-cli -g
hexo init blog  #建站初始化
cd blog
npm install
hexo server

hexo new 'title' #新建博客文章
hexo clean && hexo g && hexo s  #清理 编译 部署
